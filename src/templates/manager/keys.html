Loop: 

      You can loop through objects that are inside [] tags. For example author.
      {% verbatim %}
      {% for author in authors %}
        {{ author.first_name }}
      {% endfor %} 
      {% endverbatim %}     
      will render:

      UP

Inviting a reviewer

      In some examples, like inviting a reviewer, you will have access to that review assignment in the template so do not need to loop.
      {% verbatim %}
      {{ review.user.first_name }} {{ review.user.last_name }}
      {% endverbatim %} 
      will render:

      UP Tech
      {% verbatim %}
      {{ review.due }}
      {% endverbatim %}
      will render a date in YYYY-mm-dd format (ISO) like:

      2015-12-31
